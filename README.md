# COVID19_Global_Trends_Analysis
COVID-19 Global Trends Analysis using Python, Pandas, and Matplotlib. This project analyzes COVID-19 cases, deaths, trends, peaks, and country-level comparisons through data visualization and statistical analysis.

# COVID-19 Global Trends Analysis

## Project Overview

This project analyzes global COVID-19 trends using publicly available datasets. The analysis focuses on total cases, deaths, weekly trends, peak periods, and comparisons among Pakistan, India, and the United States.

## Objectives

* Analyze COVID-19 trends over time.
* Compare multiple countries.
* Identify patterns, peaks, and anomalies.
* Present findings through visualizations and narrative explanations.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Key Findings

* The United States recorded the highest cumulative COVID-19 cases.
* India experienced multiple significant infection waves.
* Pakistan reported comparatively lower total case counts.
* Weekly trends revealed major outbreak periods.
* Death trends generally followed infection trends.

## Conclusion

Data analysis provides valuable insights into pandemic progression and helps support evidence-based decision-making.
